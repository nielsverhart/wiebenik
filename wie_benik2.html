Waarom heb ik voor deze opleiding gekozen?
Ik heb voor deze opleiding gekozen omdat ik het leuk vind om met computers te werken en ik hier mijn creativiteit in kwijt kan.

Wat wil ik aan mezelf ontwikkelen?
Ik wil kunnen programmeren. Hiernaast wil ik ook leren hoe het beste met klanten om kan gaan want dit is voor in de toekomst erg belangrijk

Wat voor bijbanen heb ik gehad?
Ik heb op 14 jarige leeftijd bij de supermarkt gewerkt waarbij ik best veel in contact kwam met klanten. Ook heb ik enige tijd als chaffeur gewerkt in Schiedam. Dit was best een professioneel bedrijf waarbij ik eigenlijk het visite kaartje was voor onze grote klanten. Ik denk dat deze 2 baantjes aardig van pas zullen komen bij de opleiding omdat ik hier heb geleerd hoe ik mij proffesioneel moet gedragen.

Wat voor opleiding heb ik gevolgd?
Ik heb een VMBO-TL (MAVO) opleiding gevolgd en hier ook mijn diploma voor gehaald. Op deze school heb ik het erg naar mijn zin gehad en ik denk dat deze periode erg belangrijk was voor mijn ontwikkeling. Daarna heb ik de MBO-Assistent Manager Internationale Handel opleiding gevolgd aan het Albeda College in Rotterdam. Hier kwam ik er al snel achter dat deze opleiding niet bij mij past. Ik ben een poos gaan werken en uiteindelijk met deze huidige opleiding gestart.

Hoe blijf ik gemotiveerd?
Ik blijf gemotiveerd door elke dag naar school te gaan en de lessen te volgen. Ik heb nood ter benen voor deze opleiding gekozen omdat ik het leuk vind om met computers bezig te zijn.

Wat ben ik bereid om buiten de lessen te doen?
Natuurlijk is het belangrijk om buiten je lessen je huiswerk te maken. Ook kijk ik zelf al veel filmpjes over programmeren omdat ik er nog niet veel ervaring mee heb.
